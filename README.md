# Zinsontleding

### Features
* Nederlands
    * zeer beperkte woordenschat
    * substatieven
        * lidwoorden
        * meervouden
        * verkleinwoorden
    * persoonlijke voornaamwoorden
    * werkwoorden
        * overgankelijkheid
        * tegenwoordige tijd
* Zinsontleding
    * onderwerp
    * persoonsvorm
    * lijdend voorwerp
    * meewerkend voorwerp
    * congruentie controles
        * lidwoord en substantief
        * onderwerp en persoonsvorm
